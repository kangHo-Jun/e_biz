# 🚪 영림발주서 자동화 스크립트 v2 (door_v2.gs)

이 스크립트는 구글 스프레드시트의 '영림발주서' 시트에서 **단가 계산**과 **품목 코드 생성**을 자동화합니다.  
**v2 버전**은 대규모 최적화를 통해 속도가 비약적으로 향상되었으며, 모든 기능이 12행~35행 범위에서 일관되게 동작합니다.

---

## 🚀 v2 주요 변경 사항 (업그레이드)

### 1. ⚡ 압도적인 속도 향상
*   기존에는 셀 하나마다 통신하여 매우 느렸으나, **배치 처리(Batch Processing)** 기술을 적용하여 **한 번에 읽고 계산 후 한 번에 기록**합니다.
*   수십 개의 항목을 동시에 계산해도 순식간에 완료됩니다.

### 2. 📏 범위 통일 (유령행 제거)
*   모든 기능(단가계산, 코드생성, 입력초기화)이 **12행 ~ 35행** 구간에서 완벽하게 동작합니다.
*   기존에 35행이 계산되지 않던 문제가 완전히 해결되었습니다.

### 3. ✨ 멀티 행 편집 지원 (`onEdit`)
*   데이터를 여러 줄 **복사/붙여넣기** 해도 모든 줄에 대해 자동완성(색상, 가스켓 등)이 정상 작동합니다.

### 4. 🧹 스마트 초기화
*   "입력/출력 초기화" 기능이 강화되어 A열, AR~BF열뿐만 아니라 **AQ열(문짝방향, 22~35행)**도 깔끔하게 비워줍니다.

---

## 🏗️ 기능 상세 가이드

### 1. 💰 단가 계산
*   **실행**: 메뉴 > `💰 단가계산`
*   **범위**: 12행 ~ 35행 (문틀/문짝 전체)
*   **로직**:
    *   **문틀 (12~20행)**: 규격별 표준 단가 + 가스켓 옵션(+5,500원)
    *   **문짝 (22~35행)**:
        1.  표준 규격 자동 매칭 (최우선)
        2.  실패 시 A열 수동 입력값 유지 (Fallback)
        3.  AW/AX열 색상 및 키워드에 따른 추가금 자동 합산 (테스트 시트 참조)

### 2. 📦 품목 코드 생성
*   **실행**: 메뉴 > `📦 코드생성`
*   **범위**: 12행 ~ 35행
*   **출력**: BC, BD, BE, BF 열
*   **생성 규칙**:
    *   **문틀**: `브랜드(영림)` + `타입(YF/YA)` + `규격(숫자)` + `옵션(G/M)` + `색상`
    *   **문짝**: `모델(TA04 등)` + `규격(W+H+D+방향)` + `색상`

### 3. ⚡ 자동 입력 (트리거)
데이터 입력 시 실시간으로 보조 정보를 채워줍니다. (복사/붙여넣기 지원)

| 대상 열 | 자동 입력 내용 | 참조 데이터 |
| :--- | :--- | :--- |
| **AW (색상명)** | **AX (색상코드)** | 테스트 시트 V~Z열 |
| **AW (색상명)** | **BA (가스켓/단종)** | 테스트 시트 M~U열 (문틀만) |
| **AV (세로길이)** | **AQ (문짝방향)** | 테스트 시트 AG/AH열 (문짝만, H≥2166) |

---

## ⚙️ 설정 및 유지보수

### 데이터 매핑 ('테스트' 시트)
스크립트는 '테스트' 시트의 데이터를 기반으로 작동합니다.

*   **M~U열**: 품목 정보 (가스켓 색상, 단종 여부)
*   **V~Z열**: 색상 코드 매핑 (예: "영림116 (중백색)")
*   **AD~AF열**: 문짝 추가금 키워드 및 가격
*   **AG~AH열**: 문짝 방향(Y/N) 판단 키워드

### 초기화
*   메뉴 > `🧹 입력/출력 초기화` 실행
*   **삭제 범위**: A열(21행 제외), AR~BG열, AQ열(22~35행)

---

> **Tip**: 데이터가 많을 경우 `🚀 전체` 메뉴를 사용하면 **단가계산**과 **코드생성**을 한 번에 순차적으로 실행할 수 있습니다.
