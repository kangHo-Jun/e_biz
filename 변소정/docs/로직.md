# 품목명 및 데이터 추출 로직 설명

이 문서는 Google Apps Script (`Code.js`)에서 슬랙 메시지를 파싱하여 구글 시트에 입력하는 로직을 설명합니다.

## 1. 데이터 추출 개요

슬랙에서 복사된 '결제완료 안내' 메시지를 분석하여 다음 5가지 항목을 추출합니다.

| 항목 | 엑셀 열 | 추출 방식 |
| :--- | :---: | :--- |
| **날짜** | E | `YYYY-MM-DD` 형식의 날짜 패턴 매칭 |
| **요일** | F | `(월)` ~ `(일)` 형식의 요일 패턴 매칭 |
| **결제금액** | J | `결제금액:` 키워드 뒤의 숫자 추출 (콤마 제거) |
| **주문번호** | L | `주문번호` 키워드 뒤의 숫자 추출 |
| **품목명** | M | 헤더 제외 첫 번째 내용 줄에서 이름 제거 후 추출 |

---

## 2. 품목명(M열) 추출 상세 로직

가장 복잡한 품목명 추출은 다음과 같은 단계로 진행되어, 다양한 메시지 형식에도 견고하게 작동하도록 설계되었습니다.

### 단계 1: 라인 분리 및 정제
메시지를 줄 바꿈(`\n`) 기준으로 분리하고, 각 줄의 앞뒤 공백을 제거한 뒤, 내용이 있는 줄만 필터링합니다.
```javascript
const allLines = text.split('\n').map(l => l.trim()).filter(l => l.length > 0);
```
*   **효과**: 빈 줄이 섞여 있어도 실제 데이터가 있는 줄을 정확히 찾아낼 수 있습니다.

### 단계 2: 대상 라인 선정
첫 번째 줄은 항상 `[결제완료 안내]` 헤더이므로, 두 번째 줄(`allLines[1]`)을 품목명 데이터가 포함된 줄로 간주합니다.

### 단계 3: 이름 및 접두어 제거
본문 줄에서 첫 번째 하이픈(`-`)을 찾아 그 이후의 내용만 추출합니다.
```javascript
const hyphenIndex = contentLine.indexOf('-');
if (hyphenIndex !== -1) {
  item = contentLine.substring(hyphenIndex + 1).trim();
}
```
*   **패턴**: `안종혁-품목명` → `품목명` 추출
*   **특징**: 이름 앞에 이모지(`👤`)가 있거나, 이름과 하이픈 사이에 공백이 있어도 첫 번째 `-`를 기준으로 분리하므로 안전합니다.

## 3. 기타 필드 정규식

- **주문번호**: `/주문번호\s+(\d+)/`
- **금액**: `/결제금액:\s*([\d,]+)/` (추출 후 `,`를 빈 문자열로 치환)
- **날짜**: `/(\d{4}-\d{2}-\d{2})/`
- **요일**: `/\(([월화수목금토일])\)/`

## 4. 예외 처리
- `결제완료 안내]` 문구가 포함되지 않은 텍스트는 파싱 과정을 건너뜁니다.
- 데이터 입력 중 오류가 발생할 경우 `console.error`를 통해 행 번호와 에러 메시지를 기록합니다.
