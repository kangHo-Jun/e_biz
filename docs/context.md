# 프로젝트 컨텍스트 (Project Context)

이 문서는 에이전시 AI가 프로젝트의 현재 상태를 즉시 파악하고 일관된 기술적 결정을 내릴 수 있도록 돕기 위해 작성되었습니다.

## 1. 프로젝트 개요
- **프로젝트 명**: 이비즈 (e-Biz) Door System
- **목적**: 구글 시트와 Google Apps Script를 이용한 도어 주문 및 제작 관리 시스템 자동화.
- **주요 기술**: Google Apps Script (GAS), Python, Google Sheets

## 2. 현재 상태 및 완료된 기능
- **품목 타입 구분**: `구분_품목타입()` 함수를 통해 문틀, 발포, 분리형 등을 FRAME으로 분류.
- **색상 전처리**: 브랜드 색상 코드 추출 로직 최적화.

## 3. 진행 중인 것
- 목재-슬림형/와이드형 품목명 출력 오류 수정 (`구분_품목타입()` FRAME 배열에 `슬림형`, `와이드형` 추가)
- clasp push 반영 확인 중
