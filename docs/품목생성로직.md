# 품목 및 코드 생성 로직 정리

이 문서는 '문틀' 및 '문짝'의 품목명과 품목코드를 생성하는 내부 로직을 기술합니다.

## 1. 개요
영림발주서 시트의 데이터를 기반으로 ERP 업로드용 품목명(BA열)과 품목코드(BB열)를 자동으로 생성합니다.

### 대상 범위
- **문틀 (Frame)**: 12행 ~ 26행
- **문짝 (Door)**: 30행 ~ 42행

---

## 2. 입력 데이터 매핑 (Input)

시트의 각 열은 다음과 같이 매핑되어 로직에 사용됩니다.

| 열 기호 | 인덱스 | 항목 | 설명 |
| :--- | :--- | :--- | :--- |
| **AP** | 42 | 제품명 | 제품 종류 및 타입 (예: 영림ㅣ발포ㅣ와이드문틀) |
| **AQ** | 43 | 방향/옵션 | 3방/4방, 식기무/식기유 등 |
| **AS** | 45 | 규격(두께) | 문틀 두께 또는 제품 규격 |
| **AT** | 46 | 사이즈 | 가로 * 세로 (예: 880*2090) - `parseAT`를 통해 분리 |
| **AU** | 47 | 색상명 | 한글 색상 이름 (예: 영림116) |
| **AV** | 48 | 색상코드 | PS 색상 코드 (예: PS119) |

---

## 3. 품목명 생성 로직 (`생성_품목명`)

### 공통 전처리
- **색상**: `AU(색상명)`와 `AV(색상코드)`를 조합합니다. (예: "영림116 PS119")
- **접두어**: 색상명에 "영림"이 없으면 자동으로 추가합니다.

### [문틀] 품목명 생성
- **규격 조합**: `AS(두께)` + `AT(가로)` + `AT(세로)` + `방향 옵션(식기유/무)`
- **형식**: `{색상} {제품명(전처리)} {두께}*{가로}*{세로}{옵션}`
- *예시*: `영림116PS119 와이드문틀 155*880*2020식기무`

### [문짝] 품목명 생성
- **형식**: `{색상} {제품명(전처리)} {가로}*{세로}{옵션}`
- *예시*: `영림116PS119 YA-15 880*2020식기무`

---

## 4. 품목코드 생성 로직 (`생성_품목코드_NEW`)

품목코드는 `[브랜드/색상] + [모델/타입] + [규격]`의 조합으로 구성됩니다.

### ① 브랜드/색상 코드 (`브랜드색상코드_생성`)
- **Y + 숫자**: "영림116" → `Y116`
- **YS + 코드**: "PS119" → `YS119`
- **Y + 한글**: 한글 색상명 앞 2글자 (예: "더블" → `Y더블`)

### ② 모델/타입 코드
- **문틀 (`플래그코드_생성`)**:
    - **타입**: 발포(B), 방염(F), 비방염(N), 알루미늄(A)
    - **형태**: 슬림(S), 와이드(W), 분리(D), 히든(H), 미서기(L)
    - **조합 예**: 발포와이드 → `BW`
- **문짝 (`모델코드_생성`)**:
    - 제품명에서 영문 모델명 추출 (예: "YA-15" → `YA15`)

### ③ 규격 코드 (`규격코드_생성`)
- **문틀**: `두께(숫자) + 가로 + 세로 + 옵션(3방:N, 4방:Y)`
- **문짝**: `가로 + 세로`

---

## 5. 실행 및 출력
- **도구**: 시트 상단 메뉴 `📦 코드생성`을 통해 실행
- **출력 위치**:
    - **BA열 (53)**: 생성된 품목명
    - **BB열 (54)**: 생성된 품목코드
    - **BD열 (56)**: 단위 (문틀은 '틀', 문짝은 '짝')
