# 품목명 & 코드 생성 로직
**door_v4.gs | 색상_전처리 / 브랜드색상코드_생성 | 최종: 2025-03-20**

---

## 1. 품목명 생성 로직

### 1-1. 구조
```
[ 회사명 ] + [ 색상 ] + [ 품명 ] + [ 규격 ]
```
- 예시 (문틀): `영림116PS010 발포 슬림 195*900*2100식기무`
- 예시 (문짝): `영림116PS010 YA-15 880*2020식기유`

---

### 1-2. 색상 전처리 규칙 (`색상_전처리` 함수)
AU(색상명) + AV(색상코드) 조합 후 아래 패턴 순서로 처리

| 입력 패턴 | 처리 규칙 | 출력 예시 |
|----------|---------|---------|
| 영림{숫자} + 영문+숫자 | 공백 제거, 전부 출력 | 영림116 PS010 → 영림116PS010 |
| 영림{숫자} + 한글 | 한글 제거, 영림{숫자}만 출력 | 영림116 중백색 → 영림116 |
| **영림{숫자}(코드)** | **괄호 제거, 내용 유지** | **영림116(PS010) → 영림116PS010**<br>**영림147(PW951-1) → 영림147PW951-1** |
| P계열+숫자 + 한글 | 한글 제거, 영문+숫자만 출력 | PX450-3 발렌무디 다크 → PX450-3 |
| 순한글 | 공백 제거 | 시나몬 월넛 → 시나몬월넛 |

---

### 1-3. 품목명 구성

| 구성요소 | 소스 | 예시 |
|---------|-----|-----|
| 회사명 | 고정: 영림 | 영림 |
| 색상 | AU(색상명) + AV(색상코드), 색상 전처리 후 출력 | 영림116PS010 / 영림147PW951-1 / PX450-3 |
| 품명 | AP열 파싱, 문틀/문짝 키워드 추출 | 발포 슬림 / YA-15 / 와이드 |
| 규격 | AS+AT+AV열 + 식기표시(AQ열) | 195*900*2100식기무 / 880*2020식기유 |

---

### 1-4. 품명 전처리 (AP열)
- `영림ㅣ` 제거 후 `ㅣ` 구분자로 분리
- `형` 접미사 제거 (슬림형 → 슬림)
- `문틀` 키워드 제거
- 예: `영림ㅣ발포ㅣ슬림ㅣMDF` → `발포 슬림 MDF`

---

## 2. 코드 생성 로직

### 2-1. 코드 구조
```
[ 브랜드/색상코드 ] + [ 플래그/모델코드 ] + [ 규격코드 ]
```
- 예시 (문틀): `Y116BS1959002100N`
- 예시 (문짝): `Y116YA158802020`

---

### 2-2. 브랜드/색상코드 (`브랜드색상코드_생성` 함수)
영림 있으면 영림 우선, P계열은 후순위

| 순위 | 패턴 | 입력 예시 | 출력 예시 |
|-----|-----|---------|---------|
| 1 | 영림{숫자}PS{숫자} | 영림116PS010 | Y116 |
| 2 | 영림{숫자} | 영림131(PW844-1) | Y131 ← PW 부분 무시 |
| **3** | **P[A-Z]+숫자(-숫자)?**<br>**(P삭제, 나머지 영문 삽입, 하이픈 제거)** | **PS140 / PX444-1 / PSM140** | **YS140 / YX4441 / YSM140** |
| 4 | 순한글 | 시나몬월넛 | Y시나 |
| 5 | 숫자 포함 fallback | 123ABC | Y123 |

---

### 2-3. 플래그코드 (문틀)

| 상위코드 | 하위코드 | 조합 예시 | 비고 |
|---------|---------|---------|-----|
| B (발포)<br>F (방염)<br>N (비방염)<br>A (알루미늄) | S (슬림)<br>W (와이드)<br>D (분리형)<br>H (히든)<br>L (미서기) | 발포+슬림 → BS<br>분리형+와이드 → DW | 연동 있을 때:<br>F/N/A + {숫자}연동 → 상위+숫자+C<br>예: 비방염2연동 → N2C |

---

### 2-4. 모델코드 (문짝)
- 영문-{코드} 형태: 하이픈 제거
  - 예: `YA-15 → YA15` / `TA-04 → TA04`
- 탈공 → `탈`
- M/D 민무늬 → `MD`

---

### 2-5. 규격코드 (문틀)

| 구성요소 | 소스 | 예시 |
|---------|-----|-----|
| 두께 | AS열 숫자 추출 | 195바 → 195 |
| 가로 | AT열 | 900 |
| 세로 | AV열 | 2100 |
| 식기표시 | AQ열 / 3방→N / 4방→Y | 3방 → N |
| 최종 조합 | 숫자 연결 | 1959002100N |

---

### 2-6. 단위
- 문틀 → `틀`
- 문짝 → `짝`
- 레일 → `개`
